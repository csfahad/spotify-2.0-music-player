## Installation

First Fork this repo & then Clone this repo

```bash
  git clone https://github.com/<YOUR_USERNAME>/spotify-2.0-music-player.git
  cd spotify-2.0-music-player
```

Install the node_modules

```bash
  npm install
```

run the project

```bash
  npm run dev
```
